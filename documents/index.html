Documentation

The documentation below is a combination of docbook and javadocs.  The docbook is a maven module in our source tree.  If you download or check out our source code from github, you will find it under the jaxrs/docbook directory.  Javadocs are generated automatically from the maven build.

In release 3.1.0.Final, all deprecated classes in the 3.0.x generation of releases will be removed. For some guidance in upgrading, see Upgrading from Resteasy 2 to Resteasy 3.
